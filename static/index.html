WuNameAAS

Wu-Tang Clan Name Generator As A Service

Introduction

WuNameAAS (Wu-Tang Clan Name Generator As A Service) provides a modern, RESTful, scalable solution to the finding your Wu-Tang Clan Name.

API

Content Negotiation

WuNameAAS will respond to the following 'Accept:' values with appropriate content

  • text/plain - Content will be returned as a plain string.
  • application/json - Content will be returned as a JSON object { message: 'message' }
  • text/html - Content will be returned as an HTML page with a twitter bootstrap hero unit, containing the message and the subtitle.
  • application/xml - Content will be returned as a XML document.

Operations

Path Description
/wuami/:from Will return content of the form ':wuname'
For e.g. /wuami/Rahul will return 'Irate Worlock'
/enterthewu/:from Will return content of the form ':from from this day forward you will also be known as :wuname'
For e.g. /enterthewu/Rahul will return 'Rahul from this day forward you will also be known as Irate Worlock'
